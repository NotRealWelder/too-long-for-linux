# too-long-for-linux.py



Проверка длинных путей и имен файлов для Linux с учетом системных ограничений (4096/255)



Использование:  too-long-for-linux-edit.py \[-p] \[-q] \[-l ФАЙЛ] \[ДИРЕКТОРИЯ]

                too-long-for-linux-edit.py \[-h]

                python3 too-long-for-linux-edit.py \[ДИРЕКТОРИЯ]

                ./too-long-for-linux.py /etc/dir





позиционные аргументы:

  ДИРЕКТОРИЯ                    Директория для проверки (например: /etc/dir /etc/dir/ C:\\dir)



опции:

  **-h**, **--help**                    Показать это сообщение и выйти

  **-p**, **--no-progress**             Отключить индикатор прогресса

  **-q**, **--quiet**                   Тихий режим (только цифры)

  **-l**, **--log ФАЙЛ**                Сохранить результаты в указанный файл

  **--axe**                             Автоматическое исправление длинных имён файлов/директорий. Может потребовать права root. Использовать с осторожностью
