# too-long-for-Linux.py



Проверка длинных путей и имен файлов для Linux с учетом системных ограничений



Использование: 	too-long-for-linux-edit.py \[-p] \[-q] \[-l ФАЙЛ] \[ДИРЕКТОРИЯ]

&nbsp;		too-long-for-linux-edit.py \[-h]



позиционные аргументы:

&nbsp; ДИРЕКТОРИЯ			Директория для проверки



опции:

&nbsp; **-h**, **--help**			Показать это сообщение и выйти

&nbsp; **-p**, **--no-progress**		Отключить индикатор прогресса

&nbsp; **-q**, **--quiet**			Тихий режим (только цифры)

&nbsp; **-l**, **--log ФАЙЛ**		Сохранить результаты в указанный файл
